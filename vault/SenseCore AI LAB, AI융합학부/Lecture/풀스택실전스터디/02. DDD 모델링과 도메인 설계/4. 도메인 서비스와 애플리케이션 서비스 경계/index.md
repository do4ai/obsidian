# 4. 도메인 서비스와 애플리케이션 서비스 경계
이 페이지에서는 규칙을 어디서 수행할지 결정합니다. 도메인 서비스와 애플리케이션 서비스 경계가 흐리면 라우터와 서비스가 금방 비대해집니다.

# 이 페이지의 목표
- 도메인 규칙의 위치를 정한다
- 애플리케이션 서비스의 오케스트레이션 책임을 정한다
- 백엔드 구현 순서를 위한 서비스 구조를 정리한다

# 산출물
- 서비스 계층 메모
- 규칙 배치 기준
- 구현 우선순위 표

# 체크리스트
- 도메인 규칙의 위치를 정한다
- 애플리케이션 서비스의 오케스트레이션 책임을 정한다
