# 2. 사용자, 액터, 핵심 시나리오
이 페이지에서는 누가 이 서비스를 어떻게 쓰는지 정리합니다. 구매자, 운영자, AI 상담 흐름의 액터를 먼저 잡아야 기능 우선순위가 선명해집니다.

# 이 페이지의 목표
- 주요 사용자와 액터를 정의한다
- 핵심 사용 시나리오를 정리한다
- 구매자와 운영자의 목표 차이를 구분한다

# 산출물
- 액터 목록
- 핵심 시나리오 표
- 사용자 여정 요약

# 체크리스트
- 주요 사용자와 액터를 정의한다
- 핵심 사용 시나리오를 정리한다

# 다음으로 이어지는 것
다음 페이지에서는 이 시나리오를 기준으로 기능 범위와 MVP를 자릅니다.
