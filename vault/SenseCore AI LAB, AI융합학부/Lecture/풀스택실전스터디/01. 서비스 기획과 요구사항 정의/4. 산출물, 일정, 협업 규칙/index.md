# 4. 산출물, 일정, 협업 규칙
이 페이지에서는 프로젝트를 실제로 굴리기 위한 산출물 기준과 협업 규칙을 정합니다. 기획 문서, 모델링 문서, 구현, 검증이 어떤 순서로 남아야 하는지 여기서 고정합니다.

# 이 페이지의 목표
- 필수 산출물을 정리한다
- 단계별 일정과 마일스톤을 정한다
- 협업 규칙과 검증 규칙을 고정한다

# 산출물
- 산출물 체크리스트
- 마일스톤 표
- 협업 규칙 메모

# 체크리스트
- 필수 산출물을 정리한다
- 단계별 일정과 마일스톤을 정한다
