# 3. 기능 범위와 MVP 정의
이 페이지에서는 무엇을 이번 교재에서 구현하고 무엇을 뒤로 미룰지 정합니다. 범위를 먼저 고정해야 실전 서비스가 끝까지 완성됩니다.

# 이 페이지의 목표
- 핵심 기능과 후순위 기능을 구분한다
- MVP 기능 목록을 고정한다
- 챗봇과 운영 기능의 최소 범위를 정한다

# 산출물
- MVP 기능 목록
- 후속 기능 목록
- 범위 제외 메모

# 체크리스트
- 핵심 기능과 후순위 기능을 구분한다
