# 04. 개발 환경과 인프라 구축
이 단계에서는 구현 전에 개발 환경을 먼저 고정합니다. 모노레포, compose, env, DB, CI 바닥을 먼저 세워야 프론트와 백엔드가 같은 기준으로 같이 개발될 수 있습니다.

# 이 단계에서 배우는 것
- 저장소 구조와 서비스 실행 구조를 정합니다.
- 로컬 개발환경과 환경변수, 시크릿, 데이터 저장 전략을 세웁니다.
- CI와 관측의 최소 바닥을 마련합니다.
