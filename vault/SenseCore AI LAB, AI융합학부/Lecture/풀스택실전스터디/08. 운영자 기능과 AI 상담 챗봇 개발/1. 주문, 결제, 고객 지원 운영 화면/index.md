# 1. 주문, 결제, 고객 지원 운영 화면
이 페이지에서는 운영자가 주문과 결제와 고객 지원 상태를 읽는 화면을 만듭니다.

# 이 페이지의 목표
- 운영 대시보드와 목록 화면을 만든다
- 주문/결제 상태 관찰 UX를 정리한다
- 고객 지원 흐름을 운영 화면에 반영한다

# 산출물
- 운영 화면 세트
- 상태 배지 규칙
- 지원 플로우 메모

# 체크리스트
- 운영 대시보드와 목록 화면을 만든다
