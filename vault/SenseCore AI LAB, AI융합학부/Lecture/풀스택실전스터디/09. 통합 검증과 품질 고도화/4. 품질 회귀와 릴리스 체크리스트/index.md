# 4. 품질 회귀와 릴리스 체크리스트
이 페이지에서는 배포 전에 반드시 확인할 품질 기준을 정리합니다.

# 이 페이지의 목표
- 회귀 테스트 범위를 정한다
- 릴리스 전 체크리스트를 만든다
- 실패 시 보류 기준을 정한다

# 산출물
- 회귀 목록
- 릴리스 체크리스트
- 보류 기준

# 체크리스트
- 회귀 테스트 범위를 정한다
- 릴리스 전 체크리스트를 만든다
