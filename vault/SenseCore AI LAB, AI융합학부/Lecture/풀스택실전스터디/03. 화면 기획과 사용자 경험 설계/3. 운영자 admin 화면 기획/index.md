# 3. 운영자 admin 화면 기획
이 페이지에서는 운영자가 상태를 읽고 조치할 수 있는 화면을 설계합니다. 운영 화면은 구매 화면과 목적이 다릅니다.

# 이 페이지의 목표
- 대시보드와 주문, 결제 관찰 화면을 설계한다
- 상품 운영과 고객 지원 화면을 정리한다
- 권한과 역할에 따른 화면 차이를 반영한다

# 산출물
- admin 화면 목록
- 운영 플로우
- 권한별 화면 메모

# 체크리스트
- 대시보드와 주문, 결제 관찰 화면을 설계한다
- 상품 운영과 고객 지원 화면을 정리한다
