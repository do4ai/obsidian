# 2. 구매자 storefront 화면 기획
이 페이지에서는 상품 탐색부터 주문 완료까지 이어지는 구매자 화면 흐름을 설계합니다.

# 이 페이지의 목표
- 상품 목록, 상세, cart, checkout 화면을 설계한다
- 비회원과 회원 흐름 차이를 정리한다
- 오류와 빈 상태까지 화면에 반영한다

# 산출물
- storefront 화면 목록
- 사용자 플로우
- 상태별 화면 메모

# 체크리스트
- 상품 목록, 상세, cart, checkout 화면을 설계한다
- 비회원과 회원 흐름 차이를 정리한다
