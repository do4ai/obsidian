# 03. 화면 기획과 사용자 경험 설계
이 단계에서는 모델링 결과를 실제 화면 흐름으로 바꿉니다. 구매자, 운영자, AI 상담 사용자에게 각각 어떤 경험을 제공할지 먼저 설계해야 구현 순서가 자연스러워집니다.

# 이 단계에서 배우는 것
- 정보 구조와 메뉴, 라우트 맵을 정리합니다.
- storefront와 admin의 화면 책임을 나눕니다.
- AI 상담 챗봇의 대화 흐름과 UI 위치를 설계합니다.
