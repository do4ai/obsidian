# 4. 첫 통합 실행과 smoke test
이 페이지에서는 지금까지 만든 골격을 실제로 같이 띄워 봅니다. 기획과 인프라와 골격 구현이 처음으로 연결되는 순간입니다.

# 이 페이지의 목표
- 프론트와 백엔드를 함께 부팅한다
- smoke test 시나리오를 만든다
- 초기 통합 이슈를 정리한다

# 산출물
- 통합 실행 결과
- smoke test 목록
- 초기 이슈 메모
