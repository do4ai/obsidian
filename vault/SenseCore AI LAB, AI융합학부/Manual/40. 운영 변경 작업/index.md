# 40. 운영 변경 작업

이 섹션은 ingress, 도메인, 이미지 태그, 환경 변수처럼 운영 상태를 바꾸는 작업 절차를 다룬다.

변경 전 확인, 변경 적용, 변경 후 검증, 실패 시 되돌리기 기준까지 함께 정리한다.

현재 운영 기준으로 먼저 볼 문서는 아래다.

- `Ingress, 도메인, 이미지, 환경 변수 변경 절차`: GitOps 변경 전후 확인, 영향 범위 판정, 수렴 확인, 롤백 기준
